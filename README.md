# 🚀 HireTalk AI

An AI-powered voice interview platform that streamlines the hiring process by automating candidate screening, evaluation, and progression.

---

## 🌟 Features

* 🎯 **AI Voice Interviews** – Candidates answer questions using voice
* 📊 **Automated Scoring** – AI evaluates candidate responses
* 🧠 **Smart Screening** – No need for manual resume filtering
* 🗂️ **Campaign Management** – Create and manage hiring campaigns
* 🚫 **Knockout Questions** – Instantly filter candidates
* 📈 **Candidate Tracking** – Track candidates across hiring stages
* 👥 **Collaborative Hiring** – Team-based decision making
* 📅 **Live Interview Scheduling** – Conduct final interviews easily

---

## 🧩 How It Works

1. Create a hiring campaign
2. Candidates register
3. Candidates answer AI-generated questions
4. AI evaluates responses
5. Shortlist candidates automatically
6. Conduct live interviews
7. Make hiring decisions

## 🛠️ Tech Stack

* ⚡ Frontend: Vite + React

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/ashwinmali7781/HireTalk-AI.git
cd HireTalk-AI
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env` file in root:

```env
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GEMINI_API_KEY=your_gemini_api_key
```

---

### 4. Run the project

```bash
npm run dev
```

---

## 🔐 Environment Variables

Make sure you **DO NOT push `.env` file to GitHub**

Add this to `.gitignore`:

